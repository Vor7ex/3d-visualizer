# Resumen

Este visualizador 3D permite navegar en un espacio tridimensional. Consiste de tres planos perpendiculares (x, y, z). Cuenta con la opción de agregar vectores y visualizarlos en el mismo desde cualquier perspectiva.

## Mecánica de navegación

El "observador" corresponde al punto en el espacio desde donde el usuario interactuará con el mismo, inicialmente será una perspectiva isométrica centrada en el origen de coordenadas con una escala predeterminada.
La navegación de la cámara se realiza mediante un sistema de tres grados de libertad. El usuario puede controlar la orientación (giro e inclinación) mediante el arrastre con el ratón: el movimiento horizontal rota la vista lateralmente, mientras que el vertical ajusta el ángulo de elevación.
Para el tercer eje (balanceo o rotación sobre el eje óptico), se dispone de un deslizador independiente que permite una rotación sobre la profundidad, manteniendo la perpendicularidad respecto a los planos de movimiento del ratón.
Al presionar la barra espaciadora y arrastrar se ejecutará un moviemiento de traslación sin cambio de perspectiva.
